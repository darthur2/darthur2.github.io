Speech to Text¶

speech_to_text.speech_to_text.audio_transcribe(audio_file, auth_file, save_dir, prefix, language=’en-US_BroadbandModel’, watson_learn=False)[source]¶

Takes a wav audio file and uses IBM’s speech to text services to transcribe the audio into text and returns the individual words found in the audio along with names of the audio snippet files that were created for each word.

Parameters:
  • audio_file (str) – Name of location of audio file to be transcribed.
  • auth_file (str) – Name of location of .json file containing credentials needed to use the speech to text service from IBM.
  • save_dir (str) – Name of location in which audio snippets for individual words should be saved.
  • prefix (str) – Prefix to be given to each individual audio snippet file.
  • language (str) – String indicating language for transcription. See Languages for more information. Default is US English broadband.
  • watson_learn (bool) – Boolean indicating whether or not information should be sent to IBM to improve their services. Default is False.
Returns:Dictionary with individual words and the names of audio files containing the audio snippets for each word.
Return type:dict
speech_to_text.speech_to_text.create_phon_dict(cmu_dict)[source]¶

Takes the current phonetic dictionary from CMU and creates a dictionary that links each word in the dictionary to a list of phonemes for that word.

Parameters:cmu_dict (str) – File location for current CMU phonetic dictionary.
Returns:Dictionary with words as keys and lists of phonemes for each word as values.
Return type:dict
speech_to_text.speech_to_text.phoneme_matcher(audio_dict, cmu_dict)[source]¶

Creates a nested dictionary that details which individual words are in specific audio files and uses the CMU phonetic dictionary to show which phonemes are in the words that correspond to each audio file.

Parameters:
  • audio_dict (dict) – Dictionary with audio file names as keys and corresponding word values.
  • cmu_dict (dict) – CMU dictionary of words with phonemes.
Returns:

Nested dictionary matching audio file names to words and their phonemes.
Return type:

dict
speech_to_text.speech_to_text.snippet_saver(audio_file, save_dir, prefix, timestamps)[source]¶

Takes a given audio file along with the timestamps for when each word occurs in the audio and creates individual audio files for each word. Then returns names of audio files that were created.

Parameters:
  • audio_file (str) – Location of audio file.
  • save_dir (str) – Location in which audio snippets will be saved.
  • prefix (str) – Prefix to be given to each individual audio snippet file.
  • timestamps (list) – Timestamps for each individual word found in the audio.
Returns:

Names of files created.
Return type:

list
