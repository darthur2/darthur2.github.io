Speech to Text
==========================================

Speech to Text can be used to extract individual words from a wav formatted audio file. It uses the IBM Watson Developer Cloud to transcribe the audio into text. Watson also gives the user timestamps for each word. By using these timestamps, it is possible to segment the complete audio into smaller audio snippets, each of which corresponds to a single word in the transcribed audio.

Speech to Text can also be used to match up words and audio with phonemes from the CMU dictionary.
