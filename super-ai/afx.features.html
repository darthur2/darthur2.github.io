afx.features.fractal module¶

Fractal dimension features have historically been successful in discriminating sleep stages or other EEG signal classes. Here we return a feature matrix based on various definitions of fractal dimension.

afx.features.fractal.higuchi(X, kmax=8)[source]¶

Calculate the Higuchi fractal dimension. The Higuchi exponent is considered to be a measure of the irregularity of a time series. Ported from <https://www.mathworks.com/matlabcentral/fileexchange/30119-complete-higuchi-fractal-dimension-algorithm/content/hfd.m> by Salai Selvam V.

Parameters:
  • X (numpy.ndarray) – single time series.
  • kmax (int) – number of times to resample the series.
afx.features.fractal.hjorth(X, kmax=3)[source]¶

Compute Hjorth Fractal Dimension of a time series. This code was taken from pyEEG. Hjorth FD is essentially re-sampling (or decimating) the time series and checking if we have similar curves. The similarity measure is a function of the mean value for each version of the series. \(k_{\mathrm{max}}\) is the scale size (or time offset). So the algorithm compares kmax versions of the epoch for similarity. The first version skips every other datapoint, the second skips every 2 datapoints, and the kmax skips kmax datapoints at a time.

Parameters:
  • X (numpy.ndarray) – single time series.
  • kmax (float) – basically the scale size or time offset; creates kmax versions of your time series. The k-th series is every k-th time of the original series.
afx.features.fractal.hurst(X)[source]¶

Returns the Hurst Exponent of the time series vector. Hurst is a measure of long-term memory in a time series that is not due to periodicity. It’s often used as a measure how how stationary a time series is.

Parameters:X (numpy.ndarray) – single time series.
Returns:
the rate at which the autocorrelation in a time series
decreases as the lags increase. If the series \(x(t)\) is a self-similar fractal, then \(x(bt)\) is statistically equivalent to \(b^Hx(t)\), where H is the Hurst exponent.
Return type:float
afx.features.fractal.katz(X)[source]¶

Calculate the Katz fractal dimension. Example.

Parameters:X (numpy.ndarray) – single time series.
Returns:
the Katz fractal dimension; \(\frac{\log(L)}{\log(d)}\), where \(L\)
is extent of first point to furthest point, \(d\) is length of time series.
Return type:float
afx.features.fractal.petrosian(X, D=None)[source]¶

Compute Petrosian Fractal Dimension. Petrosian’s algorithm translates the series into a binary sequence (a vector of +1 and -1 values). \(\frac{\log_{10}(n)}{\log_{10}(n) + \log_{10}(\frac{n}{n + 0.4 N_\Delta})}\). This code was adapted from pyEEG, 0.02 r1: http://pyeeg.sourceforge.net/ .

Parameters:
  • X (numpy.ndarray) – single time series.
  • D (numpy.ndarray) – first order differential sequence of X; if not provided, D is computed by numpy.diff().
Returns:

the Petrosian fractal dimension.
Return type:

float

afx.features.stt.snippet_saver(audio_file, word_times, target_directory)[source]¶

Uses the information provided by the word_times function to segment an audio file into individual words and saves the files.

Parameters:

  • word_times (dict) – Dictionary from word_times function containing information about words and when they occur in audio_file.
  • target_directory (str) – Directory in which the resulting audio snippets are to be saved.
Returns:

Nothing. Saves audio files with words. Names of each file are determined by the words they contain.

afx.features.stt.word_times(audio_file, auth_file, language=’en-US_BroadbandModel’, watson_learn=False)[source]¶

Takes a wav audio file and uses the IBM Watson Speech to Text API to transcribe the audio into text and then extracts words with timestamps.

Parameters:

  • auth_file (str) – Location of json file that contains Bluemix credentials needed by IBM to transcribe audio.
