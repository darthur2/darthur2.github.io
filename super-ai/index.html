Automatic Feature Extractor Documentation¶

The super-ai project is, unfortunately, not a solution to the problem of the general artificial intelligence, even if its name may suggest that. Instead, it is a framework that allows automated feature extraction (hence the python project name afx) from arbitrary datasets. The overview below describes the general approach.

Note

Building ML models is a multi-step process. This project and documentation does not deal with any actual model building! Instead, it relies on the generic model framework and algorithm selection projects such as the projects listed at automl and automl python. Feature selection is the dark art of data science since model selection and hyper-parameter optimization are essentially solved problems: with computer-selected models and parameters already outperforming humans. As such, this project aims at automating the feature selection and leaving model and hyper-parameter selection to other existing projects.
