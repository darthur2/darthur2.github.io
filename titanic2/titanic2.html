Titanic2 Module¶

The titanic2 package includes a suite of functions that use training data sets for both survivors and non-survivors of the Titanic to perform feature engineering, fit a random forest model, and make predictions for a test data set.

titanic2.titanic2.Extract_Features(file_name)¶

Reads in the Titanic csv file and extracts features needed to make predictions.

Parameters:file_name (str) – Location of csv file that contains variables for the Titanic analysis.
Returns:Data frame of features to be used in training a random forest model.
Return type:pandas.DataFrame 
file_name = '~\train.csv'
train_data = Extract_Features(file_name)
titanic2.titanic2.Fit_Random_Forest(features, train_file, n_est=1000, max_feat=’sqrt’, min_samp=10, min_leaf=2)¶

Takes the response from the training data and the features from the training data and trains a random forest model and then returns that model for future predictions.

For further details on appropriate values for n_est, max_feat, min_samp, and min_leaf or to see default values used when when training the random forest model, see information on RandomForestClassifier in the scikit-learn documentation.

Parameters:
  • features (pandas.DataFrame) – Features used to train random forest for future predictions.
  • train_file (str) – Location of file containing training data.
  • n_est (int) – Number of trees to be used in random Forest (changed from default of 10).
  • max_feat (int, float, or str) – Maximum number of features to split on at each node (changed from using all features).
  • min_samp (int, float) – Minimum number of samples needed to split at internal node (change from default of 2).
  • min_leaf (int, float) – Minimum number of samples needed at each leaf node (changed from default of 1).
Returns:

Trained model to be used for predictions.
Return type:

sklearn.ensemble.RandomForestClassifier
titanic2.titanic2.Predict_File(rf_model, test_file, out_file_name)¶

Takes trained random forest model and uses it to make predictions using the features from data in test data set. Then it outputs the predictions for each passenger in the test data set along with their passenger ID.

Parameters:
  • rf_model (sklearn.ensemble.RandomForestClassifier) – Trained model to be used for predictions.
  • test_file (str) – Name of file location for test data set.
  • out_file_name (str) – Name of file location for output file containing predictions.
Returns:

Nothing. Creates a file.
