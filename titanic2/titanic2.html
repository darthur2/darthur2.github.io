The titanic2 package includes a suite of functions that use training data sets for both survivors and non-survivors of the Titanic to perform feature engineering, fit a random forest model, and make predictions for a test data set.

titanic2.titanic2.Extract_Features(file_name)¶

Reads in the Titanic data set found in the csv file located in file_name as a pandas data frame and extracts features needed to fit a Random Forest model to make predictions for survivors and non-survivors of the Titanic.

The features needed for the fitted model come from the following variables:

Age - Age of passenger Fare - Fare paid by passenger Sex - Gender of passenger Pclass - Assigned social class for passenger SibSp - Number of siblings/spouses aboard Titanic Name - Name of Passenger

In addition, the variable Parch (number of parents/children aboard Titanic) is needed to fill in missing values for Age.

Args:
file_name (str): Location of csv file that contains variables for the Titanic analysis.
Returns:
pandas.DataFrame: Data frame containing extracted features used for fitting the Random Forest needed to make predictions.
titanic2.titanic2.Fit_Random_Forest(features, train_file)¶

Takes the response from the training data and the features from the training data and trains a Random Forest model and then returns that model for future predictions.

Args:
features (pandas.DataFrame): Features used to train Random Forest for future predictions. train_file (str): Location of file containing training data.
Returns:
rf_model (sklearn.ensemble.RandomForestClassifier): Trained model to be used for predictions.
titanic2.titanic2.Predict_File(rf_model, test_file, out_file_name)¶

Takes the trained Random Forest model (rf_model) and uses it to make predictions using the features from test_data. Then it outputs the predictions for each passenger in the test data set along with their passenger ID.

Args:
rf_model (sklearn.ensemble.RandomForestClassifier): Trained model to be used for predictions. test_file (str): Name of file location for test data set. out_file_name (str): Name of file location for output file containing predictions.
Returns:
Nothing. Creates a file.
